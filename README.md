# 🤖 AI 3D Learning System

An interactive AI-powered application that generates explanations for objects and simulates avatar actions using natural language.

🔗 **Live Demo:** https://lordlethabo-ai-3d-learning.hf.space

---

## 📌 Overview

This project demonstrates how AI can be used to:
- Interpret natural language input
- Generate contextual explanations
- Simulate interactive avatar commands
- Provide a simple 3D learning interface

The system is designed as a prototype for AI-driven educational and interactive platforms.

---

## ✨ Features

### 🔹 AI → 3D Object Generator
- Input: Any object (e.g. *helmet, car, robot*)
- Output:
  - AI-generated explanation
  - 3D model reference (GLB)

### 🔹 Avatar Control System
- Input: Natural language command (e.g. *walk, wave*)
- Output:
  - Interpreted action
  - Simulated avatar response

---

## 🧠 How It Works

1. User enters text input
2. Input is sent to an AI model via API
3. AI processes and returns:
   - Explanation OR action
4. Frontend displays results in real-time
